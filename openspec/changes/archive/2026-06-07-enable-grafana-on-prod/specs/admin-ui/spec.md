## ADDED Requirements

### Requirement: Integrated Grafana Link
The Admin UI SHALL provide a prominent link or navigation item to access the Grafana telemetry visualization dashboards.

#### Scenario: Navigating to Grafana from Admin UI
- **WHEN** an administrator is on the Admin UI dashboard
- **THEN** they see a link to "Analytics" or "Grafana"
- **AND** clicking it takes them to `https://your-domain.com/grafana`
